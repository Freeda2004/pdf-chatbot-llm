\# 📄 PDF Chatbot using LLM



This project is a chatbot that answers questions from uploaded PDF documents using Large Language Models (LLMs).



\## 🚀 Features

\- Upload PDF

\- Ask questions about the document

\- Uses embeddings and vector search

\- Built with Streamlit and LangChain



\## 🛠 Tech Stack

\- Python

\- Streamlit

\- LangChain

\- FAISS

\- HuggingFace Transformers



\## ▶ Run the Project



1\. Create virtual environment

2\. Install dependencies

3\. Run



streamlit run app.py
